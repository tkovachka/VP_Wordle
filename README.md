## 1. Опис на проектната 

Играта што ја имплементираме е со истата идеја како постоечката Wordle (Линк: https://www.nytimes.com/games/wordle/index.html). Целта е во 5 обиди и за најмногу 5 минути, корисникот да погоди збор избран по случаен избор. 
Во нашата имплементација, разликата е тоа што корисникот има можност да избере дали ќе погодува збор од 5,6 или 7 букви.

## 3. Опис на проблемот преку податоци, класи, функции

### Класа Dictionary 
- се чуваат 3 листи (со зборови од 5, 6 и 7 букви) од кои се бира некој збор на Random позиција

### Класа Square
- означува едно квадратче во кое се внесува буква
- се чува променлива Status:int за неговата состојба на почеток поставена на 0 (0=бело, 1=жолто, 2=зелено, 3=сиво)
- Draw метод кој во зависност од состојбата на квадратчето го бои во потребната боја
- метод за додавање на буква во квадратчето - void:AddLetter(string letter)

### Класа Word
- List<Square> Squares - се чува листа од квадратчиња, претставува еден ред во формата 
- променлива bool IsFull, иницијално поставена на false, која станува true кога ќе се исполни целата листа со букви (ред)
- void Draw(Graphics g) метод во кој се повикува Draw за сите квадратчиња во редот
- void addLetter(string letter), во кој се додава буква во следното празно квадратче во редот, а доколку е последно, ја сетира променливата IsFull на true 
- public void checkLetters(string word) - при клик на ентер се повикува методот, ги проверува буквите со помош на менување на состојбата на сите квадратчиња во зависност од тоа дали се наоѓат во зборот и каде (имплементација и за дупликати на букви, односно доколку буквата се наоѓа во дадениот збор помалку пати отколку во погодениот збор, се бои само колку пати што ја има, а не повеќе)
- public bool IsCorrect() - враќа true само ако состојбата на сите квадратчиња е 2 (зелена боја)

### Класа Scene 
- List<Words> - се чува листа од 5 "зборови" кои ги претставуваат редовите
- Dictionary dictionary - објект од класата Dictionary кој се одбира со помош на Random објект во конструкторот на класата во зависност од избраниот број на букви
- bool GameOver - се сетира на true при искористување на сите обиди/изминување на времето
- променлива bool full - станува true кога моменталниот ред во кој пишуваме е исполнет, а во спротива е false
- List<string> lettersGuessed - букви кои се погодувани 
- public string lettersLeft() со кој се одредуваат преостанатите букви за погодување (кои се наоѓаат во lettersGuessed)
- public bool Check() - го повикува методот checkLetters() за секој Word што е исполнет, а доколку стигнеме до последниот ред, ја променува вредноста на GameOver во true. Враќа true/false во зависност од тоа дали моменталниот збор кој се проверува е точен или не

### Форма Intro
- се справува само со клик на копчето Start, со тоа што ја отвара формата Wordle во која се наоѓаат сите податоци, а самата таа се затвара

### Форма Wordle
- Scene scene - објект од сцената
- int TimeLeft - поставено на 300 при иницијализирање на формата 
- Paint метод во кој се повикува функцијата Draw од сцената
- private void Form1_KeyDown(object sender, KeyEventArgs e) - при притиснување на копче од тастатура - испишана буква (додавање на буквата во сцената), копче "Enter"(проверување на зборот) или "Backspace"(бришење на буква)
- private void timer1_Tick(object sender, EventArgs e) - намалување на вредноста на TimeLeft за 1, ажурирање на содржината во текстот за преостанато време

 public void checkLetters(string word)
        {
            string guessedWord = GuessedWord(Squares);

            List<string> duplicateLettersInWord = DuplicateLetters(word); // листа од буквите кои се дупликати во зборот кој се погодува
            List<string> duplicateLettersInGuessed = DuplicateLetters(guessedWord); // листа од букви кои се дупликати во напишаниот збор од корисникот

            List<GuessedLetter> duplicatesInWord = FillList(word); // листи од објекти од класата GuessedLetter (ја чува буквата и колку пати се појавува во зборот кој се бара)
            List<GuessedLetter> duplicatesInGuessed = FillList(guessedWord); 

            // на почеток ја менува состојбата на сите квадратчиња во сива боја
            foreach (Square s in Squares)
            {
                s.ChangeStatus(3);
            }

            // го споредува напишаниот збор со дадениот и на оние букви кои се на вистинско место ја поставува состојбата на зелена боја
            for (int i = 0; i < Squares.Count; i++) 
            {
                if (Squares[i].Letter == word[i].ToString())
                {
                    Squares[i].ChangeStatus(2);
                }
            }

            // ја споредува секоја буква од квадратчињата со секоја од зборот 
            for (int i = 0; i < Squares.Count; i++)
            {
                for (int j = 0; j < word.Length; j++)
                {
                    if (Squares[i].Letter == word[j].ToString()) // доколку се најде буква што е иста како во зборот 
                    {
                            if (duplicateLettersInWord.Contains(Squares[i].Letter)) // се проверува дали буквата е дупликат во зборот кој се погодува 
                            {
                                foreach (GuessedLetter l in duplicatesInWord) // доколку е дупликат, ја наоѓаме во листата како објект GuessedLetter
                                {
                                    if (l.Letter == Squares[i].Letter)
                                    {
                                        if (CanWrite(Squares[i].Letter, word, guessedWord, i)) // доколку има повеќе појавувања на буквата во зборот околку во 
         нашиот збор досега, буквата може да се обои жолта (ова се проверува во методот bool:CanWrite(string,string,string,int))
                                        {
                                            Squares[i].ChangeStatus(1);
                                        }
                                    }
                                    }
                        }
                        else // доколку буквата не е дупликат во зборот кој се погодува
                        {
                            if (duplicateLettersInGuessed.Contains(Squares[i].Letter)) // но, буквата е дупликат во погодениот збор на клиентот (смее да се обои само еднаш во зборот)
                            {
                                bool green = false;
                                foreach(Square s in Squares) // проверуваме дали веќе е погодена
                                {
                                    if (s.Status == 2)
                                    {
                                        green = true;
                                    }
                                }

                                if (!IsGuessed(Squares[i].Letter,guessedWord,i)&&!green) // доколку буквата не била погодена, ниту е обоена во жолто, може да се обои
                                {
                                    if(i!=j)
                                        Squares[i].ChangeStatus(1);
                                }
                            }
                            else
                            {
                                if(i!=j) // не е дупликат во ниеден збор - се бои жолта
                                Squares[i].ChangeStatus(1);
                            }
                        } 
                    }
                }
            }
        }
